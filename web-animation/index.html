Model Building under Earthquake Simulator

This simulation uses data extracted from the python file. The data encapsulates the frequency response (both phase and magnitude). The frequency range is between 0 and 20Hz, which is similar to the range of an actual earthquake. Try and find the resonant frequency. As this is a 3-body problem, there are actually three resonant frequencies. However, the fundamental resonant frequency is much more obvious than the higher harmonics. There have been adaptations to the model to show a more obvious progression of a resonant frequency. The model is also being played at a quarter of the actual frequency to show the effects more clearly.
